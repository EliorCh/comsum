ОъОћ ОћОЎОћ ОюОаОЋ ОЉОЕОЏОЉОф ОћОфОбОЉОЋОеОћ

ОЕОЎОЕОћ ОцОеОДОЎОЮ ОЉОЊОДОћ ОљОЌОф Рђћ ОъОЕОюОЎОЌОћ ОбОЎОЋОЋОеОф ОЉ-UDP ОЋОбОЊ ОЉОДОеОф ОњОЋОЊОЕ ОъОюОљОћ ОЉ-TCP.

­ЪЌ║№ИЈОъОцОф ОћОЕОЏОЉОћ
ОцОеОДОЕОЋОеОћ ОфОЌОфОЋОаОћ
2.1 ┬и UDP ОЋОъОЋОюОўОЎОцОюОДОАОЎОаОњОфОбОЉОЋОеОћ ОъОЎОаОЎОъОюОЎОф: ОЉОюОЎ ОЌОЎОЉОЋОе, ОЉОюОЎ ОбОеОЉОЋОЎОЋОф, ОъОћОЎОе. ОцОЋОеОўОЎОЮ = ОъОЋОюОўОЎОцОюОДОАОЎОаОњ ОЉОЎОЪ ОљОцОюОЎОДОдОЎОЋОф ОбОю ОљОЋОфОЋ ОъОЌОЕОЉ.
2.2 ┬и ОћОбОЉОеОћ ОљОъОЎОаОћ (RDT)ОљОЎОџ ОЉОЋОаОЎОЮ ОљОъОЎОаОЋОф ОъОбОю ОбОеОЋОЦ ОюОљ ОљОъОЎОЪ: ACKРђЈ, ОъОАОцОеОЎОЮ ОАОЎОЊОЋОеОЎОЎОЮ, ОўОЎОЎОъОеОЎОЮ Рђћ ОдОбОЊ-ОдОбОЊ Оъ-rdt1.0 ОбОЊ rdt3.0.
2.3 ┬и TCP Рђћ ОъОЉОаОћОћОАОњОъОаОў: РђЈSEQ ОюОцОЎ ОЉОЎОЎОўОЎОЮ (ОюОљ ОАОњОъОаОўОЎОЮ), РђЈACK = ОћОЉОЎОЎОў ОћОЉОљ ОЕОъОдОцОЎОЮ ОюОЋ. ОЌОЎОЕОЋОЉ Timeout Оъ-EstimatedRTT.
2.4 ┬и ОЉОДОеОф ОќОеОЎОъОћОюОљ ОюОћОўОЉОЎОб ОљОф ОћОъОДОЉОю: РђЈRW = ОЏОъОћ ОъОДОЋОЮ ОаОЕОљОе ОЉОЉОљОцОе ОЕОюОЋ. ОћОЕОЋОюОЌ ОюОљ ОЌОЋОеОњ ОъОъОаОЋ.
2.5 ┬и ОћОДОъОф ОЋОАОњОЎОеОф ОДОЕОеРђЈ3-Way HandshakeРђЈ (SYN Рєљ SYN-ACK Рєљ ACK) ОюОцОфОЎОЌОћ, ОљОеОЉОбОћ ОъОАОеОЎОЮ (FIN/ACK ├Ќ2) ОюОАОњОЎОеОћ ОъОАОЋОЊОеОф.
2.6 ┬и ОЉОДОеОф ОњОЋОЊОЕОюОљ ОюОћОўОЉОЎОб ОљОф ОћОеОЕОф: РђЈSlow Start ОъОбОеОЎОЏОЎ ОбОЊ ssthresh, ОљОќ AIMD ОюОЎОаОЎОљОеОЎ. Timeout ОъОљОцОА, РђЈ3 Dup ACKs ОЌОЋОдОћ.
­ЪЊљОћОаОЋОАОЌОљОЋОф ОЕОю ОћОЕОЏОЉОћ
  • Timeout: $\text{EstimatedRTT} + 4 \cdot \text{DevRTT}$, ОЏОљОЕОе РђЈEstimatedRTT ОћОЋОљ ОъОъОЋОдОб ОаОб: $(1-\alpha)\cdot\text{Est} + \alpha\cdot\text{Sample}$ РђЈ($\alpha = 0.125$).
  • ACK: РђЈSeq + Length ОЕОю ОъОћ ОЕОћОфОДОЉОю (Seq=90, ОљОЋОеОџ=40 Рєљ ACK=130).
  • RW: РђЈRcvBuffer Рѕњ (LastByteRcvd Рѕњ LastByteRead). ОћОЕОЋОюОЌ ОЕОЋОъОе: ОюОљ-ОъОљОЋОЕОе РЅц RW.
  • Reno ОбОю ОљОЎОеОЋОб ОњОЋОЊОЕ: РђЈTimeout Рєљ РђЈssthresh = cwnd/2, cwnd = 1РђЈ; РђЈ3 Dup ACKs Рєљ РђЈssthresh = cwnd/2, cwnd = ssthresh ОЋОъОъОЕОЎОЏОЎОЮ ОЉ-AIMD.
Рџа№ИЈОўОбОЋОЎОЋОф ОаОцОЋОдОЋОф ОЉОъОЉОЌОЪ
  • RW РЅа CW. ОЉОДОеОф ОќОеОЎОъОћ ОъОњОЎОаОћ ОбОю ОћОъОДОЉОю, ОЉОДОеОф ОњОЋОЊОЕ ОбОю ОћОеОЕОф. ОћОЌОюОЋОЪ ОЉОцОЋОбОю = ОћОъОЎОаОЎОъОЋОЮ ОЉОЎОЪ ОЕОаОЎОћОЮ.
  • РђЈSEQ ОАОЋОцОе ОЉОЎОЎОўОЎОЮ, ОюОљ ОћОЋОЊОбОЋОф Рђћ ОюОЌОЕОЉ ACK ОюОцОЎ ОъОАОцОе ОАОњОъОаОўОЎОЮ ОќОЋ ОўОбОЋОф ОДОюОљОАОЎОф.
  • РђЈTahoe ОъОЋОю Reno: РђЈTahoe ОъОљОцОА Ою-1 ОњОЮ ОбОю 3 Dup ACKsРђЈ; Reno ОЌОЋОдОћ ОЉОюОЉОЊ (Fast Recovery).
  • РђЈssthresh ОљОБ ОцОбОЮ ОюОљ ОбОЋОюОћ Рђћ ОћОЋОљ ОеОД ОаОЌОфОџ ОЉОЌОдОЎ. ОћОњОЊОЎОюОћ ОъОЌОЊОЕ ОћОЎОљ ОЕОю cwnd.
